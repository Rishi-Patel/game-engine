# A miniature game engine created by patelris

Note: This requires OpenGL and for Linux users the X11 dev packages:
```
sudo apt install mesa-common-dev libgl1-mesa-dev
sudo apt install libxext-dev libx11-dev libxrandr-dev libxcursor-dev libxi-dev
```

Linux also requires sound headers:
```
sudo apt-get install libasound2-dev
OR
sudo apt-get install libpulse-dev
```
